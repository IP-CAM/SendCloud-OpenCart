GitHub README info

Deze extensie stelt je in staat om orders vanuit OpenCart 2 naar SendCloud te transporteren.
Vervolgens kun je de bestellingen verwerken vanuit het SendCloud platform.

Deze extensie is officieel ondersteund vanuit SendCloud BV

Voel je vrij om mee te helpen aan de verdere ontwikkeling van deze module.

.. include:: ../../README.rst
	:start-after: compatibility of this extension.
	:end-before:  Version Support

Versie ondersteuning
====================
Deze Extensie is bedoeld voor OpenCart 2.0.

Een niet open-source versie van de extensie voor OpenCart 1.5.x kan gedownload worden in de OpenCart Extensie winkel

.. include:: ../../README.rst
	:start-after: downloaded from the OpenCart extension store:
	:end-before: Upgrading

Je webwinkel updaten naar OpenCart 2 is aangeraden om te kunnnen profiteren van de laatste updates.
